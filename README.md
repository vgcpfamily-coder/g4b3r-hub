# g4b3r-hub

Este repositório contém scripts e módulos para automação e gerenciamento de funcionalidades no Roblox, focado principalmente em sistemas de frutas, farm, notificações e configurações.

## Estrutura
- **DataBase.obfuscated.lua**: Banco de dados principal.
- **load/**: Scripts de carregamento e testes.
- **modules/**: Módulos de automação (dropar, guardar, randomizar frutas, farm, notificações, etc.)
  - **settings/**: Configurações específicas (ant-afk, auto-haki, remover luz, etc.)
- **sections/**: Scripts de seções (farm, frutas, configurações).
- **tela/**: Scripts de interface (dropdown, alertas).

## Como usar
1. Clone o repositório:
   ```bash
   git clone https://github.com/vgcpfamily-coder/g4b3r-hub.git
   ```
2. Utilize os scripts conforme a necessidade no ambiente Roblox.

### Carregue diretamente no Roblox:

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/vgcpfamily-coder/g4b3r-hub/refs/heads/main/load"))()
```
